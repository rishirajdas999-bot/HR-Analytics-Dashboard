# HR Analytics Dashboard

## 📌 Overview

This project analyzes employee attrition using Power BI.

The dashboard helps understand:

- Employee turnover by department
- Employee turnover by salary
- Satisfaction levels
- Years spent in the company
- Average monthly working hours

---

## 🛠 Tools Used

- Microsoft Power BI
- Microsoft Excel

## 📈 Key Insights

- Employees with low salaries have the highest attrition.
- Sales department has the highest employee turnover.
- Lower employee satisfaction is associated with higher attrition.
- Most employees leave after spending several years in the company.
